[Risostack](https://corndogs.github.io/risostack/) is a risograph-inspired web tool for converting images into customizable, physically-accurate ink passes. 

[Risograph printing](https://stencil.wiki/wiki/Risograph) is a fast, eco-friendly, rapid image duplication process used for making vibrant prints, posters, and zines at low-cost. The process is often described as a cross between screen printing and photocopying. It's more resource-friendly, and is better for the environment.

You can find your nearest risograph [here](https://stencil.wiki/wiki/An_Atlas_of_Modern_Risography).
